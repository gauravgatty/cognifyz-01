# 🚀 Smart Form App (Node.js + Express + EJS)

A simple full-stack web application that demonstrates server-side rendering and form handling using Node.js.

## ✨ Features
- User input form
- Server-side processing with Express
- Dynamic rendering using EJS
- Clean UI with modern design

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- HTML/CSS
